## Inspiracje

**Słowa, skojarzenia, pomysły:**
- Porządek, rutyna, małe kroki, prostota
- Gamifikacja, motywacja, system nagród
- Przypomnienia, Pomodoro, tryb skupienia
- Kolorowe kategorie, szybkie dodawanie, minimalny UI
- Widok tygodniowy, checklisty, szablony zadań

---

## Elementy projektu

- Lista zadań (dodaj / edytuj / usuń)
- Kalendarz tygodniowy / widok dnia
- Powiadomienia i przypomnienia
- Proste statystyki (czas spędzony, ukończone zadania)
- Kategorie / tagi / priorytety
- Szablony zadań i zadania powtarzalne
- Tryb skupienia (Pomodoro)
- Ustawienia użytkownika
- Eksport / import danych

---

## Wstępne dane wejściowe

**Kategorie:**
- Nauka
- Zajęcia
- Projekty
- Przygotowania
- Egzaminy

**Pola zadania:**
- Tytuł
- Opis
- Data
- Czas trwania
- Kategoria
- Priorytet
- Powtarzanie
- Notatki

**Funkcje:**
- Dodaj / edytuj / usuń zadanie
- Ustaw przypomnienie
- Oznacz jako wykonane
- Filtrowanie i sortowanie

**Opcjonalnie:**
- Synchronizacja
- Eksport do CSV

---

## Źródła pomysłu

- Własna potrzeba – brak struktury w nauce
- Znajomi ze studiów – feedback o przeładowaniu informacjami
- Istniejące narzędzia jako inspiracja:
  - Todoist
  - Google Calendar
  - Notion
- Artykuły o produktywności i technikach pracy:
  - Pomodoro
  - Time blocking
- Przypadki użycia:
  - Przygotowania do egzaminów
  - Organizacja projektów semestralnych
