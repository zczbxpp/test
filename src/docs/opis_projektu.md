# Opis projektu

Projekt dotyczy aplikacji pomagającej studentom planować naukę.  
Rozwiązuje problem braku struktury i przeładowania informacjami na studiach.

## Odbiorcy
- Osoby w wieku **18–25 lat**
- Studenci, którzy chcą lepiej organizować zadania

## Funkcjonalności
- Lista zadań
- Kalendarz tygodniowy
- Proste statystyki
